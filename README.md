## Usage

1. Open `.tex` file
2. Click preview icon or use Command Palette
3. Edit and save - preview updates automatically

## Commands

- `LaTeX: Show Preview` - Open PDF preview
- `LaTeX: Refresh Preview` - Manual refresh
